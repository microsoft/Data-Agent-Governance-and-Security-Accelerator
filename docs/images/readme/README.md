# README Section Header Images

This folder should contain the section header images for the main README.md file, following the GSA (Global Solution Accelerator) pattern.

## Required Images

Create the following images (recommended size: 32x32 or similar small icon):

| Filename | Section | Description |
| -------- | ------- | ----------- |
| `solution-overview.png` | Solution overview | Icon representing architecture/overview |
| `quick-deploy.png` | Quick deploy | Icon representing deployment/rocket |
| `business-scenario.png` | Business scenario | Icon representing business/use case |
| `supporting-documentation.png` | Supporting documentation | Icon representing documentation/books |

## Style Guidelines

- Use consistent icon style across all images
- Recommended: Azure-style icons or simple line icons
- Keep file sizes small for fast loading
- PNG format with transparent background preferred
